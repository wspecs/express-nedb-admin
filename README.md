## Install

`npm install --save express-nedb-admin`

Or as binary

``npm install --g express-nedb-admin`

## Scripts

 - **npm run build** : `rm -rf dist && tsc && npm run readme`
 - **npm run readme** : `node ./node_modules/.bin/node-readme`
 - **npm run test** : `mocha ./dist/test/*.js`
